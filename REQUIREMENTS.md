# WB ERP 产品需求文档

> 整理时间：2026-03-20
> 项目目标：俄罗斯电商平台 Wildberries 日常销售管理工具

---

## 项目背景

针对俄罗斯电商平台 Wildberries 的日常销售管理工具，部署在云服务器上，通过浏览器访问。

## 技术栈

- **后端**：Python + FastAPI
- **前端**：Vue 3 + Element Plus
- **数据库**：SQLite（简单）/ PostgreSQL（生产）
- **部署**：Docker + Nginx
- **服务器**：腾讯云 2核4G Ubuntu 24.04

---

## 核心功能需求

### 1. 销售数据看板

**功能描述**：实时显示销售数据概览

**筛选条件**：
- 时间：今日 / 昨日 / 7天 / 30天 / 自定义时间
- 店铺：单选 / 多选
- 负责人：单选 / 多选
- 产品：单选 / 多选

**数据指标**：
| 指标 | 说明 |
|------|------|
| 销售额 | 订单总金额 |
| 订单量 | 订单数量 |
| 访客 | 访问人数 |
| 加购 | 加入购物车次数 |
| 加购率 | 加购 / 访客 |
| 转化率 | 订单 / 访客 |
| 广告费 | 广告花费总额 |
| 广告占比 | 广告费 / 销售额 |

**API 对接**：
- `GET /api/v1/supplier-report` - 销售报表
- `GET /api/v1/report/sales` - 销售统计

---

### 2. 广告分析

**功能描述**：分析每个产品的广告效果

**筛选条件**：与销售数据看板相同

**数据指标**：
| 指标 | 说明 |
|------|------|
| CPM 搜索 | 搜索广告千次展示成本 |
| CPM 推荐 | 推荐广告千次展示成本 |
| CPC 搜索 | 搜索广告点击成本 |
| 展示量 | 广告展示次数 |
| 点击量 | 广告点击次数 |
| 点击率 | 点击 / 展示 |
| 广告花费 | 总花费 |
| 广告销售额 | 广告带来的销售额 |
| ACOS | 广告花费 / 广告销售额 |
| ROAS | 广告销售额 / 广告花费 |

**优化建议**：
- ACOS 过高 → 降低出价或优化关键词
- 点击率低 → 优化主图/标题
- 转化率低 → 优化详情页/价格

**API 对接**：
- `GET /adv/v1/promotion/adverts` - 广告列表
- `GET /adv/v1/stat` - 广告统计

---

### 3. 产品管理

**功能描述**：从平台导入产品信息，支持自定义编辑

**产品字段**：
| 字段 | 来源 | 可编辑 |
|------|------|--------|
| 产品ID | WB API | ❌ |
| SKU | WB API | ❌ |
| 店铺 | 用户选择 | ❌ |
| 产品名称 | WB API → 可自定义 | ✅ |
| 负责人 | 用户设置 | ✅ |
| 重量 | 用户设置 | ✅ |
| 长度 | 用户设置 | ✅ |
| 宽度 | 用户设置 | ✅ |
| 高度 | 用户设置 | ✅ |

**同步规则**：
- 每次同步只新增，不覆盖已有数据
- 已存在的产品ID跳过

**API 对接**：
- `GET /content/v2/get/cards/list` - 获取商品列表

---

### 4. 库存管理

**功能描述**：管理产品库存，记录成本

**库存来源**：
| 来源 | 说明 |
|------|------|
| FBW | WB 仓库库存（API 拉取） |
| FBS | 卖家仓库库存（API 拉取） |
| 自有仓库 | 手动录入 |

**入库记录**：
| 字段 | 说明 |
|------|------|
| 产品 | 关联产品ID |
| 数量 | 入库数量 |
| 产品成本 | 单件成本（元/卢布） |
| 物流成本 | 物流费用 |
| 入库时间 | 记录时间 |
| 仓库类型 | FBW/FBS/自有 |

**成本核算**：
- 先进先出（FIFO）原则
- 每次出库按最早入库成本计算
- 为财务计算做准备

**API 对接**：
- `GET /api/v3/warehouses` - 获取仓库
- `POST /api/v3/stocks/{warehouseId}` - 查询库存

---

### 5. 利润分析

**功能描述**：自动计算每单实际利润

**利润计算公式**：
```
利润 = 销售额 - 平台佣金 - 物流费 - 产品成本 - 广告费 - 其他成本
```

**数据来源**：
| 项目 | 来源 |
|------|------|
| 销售额 | WB API（订单） |
| 平台佣金 | WB API（订单详情） |
| 物流费 | WB API（订单详情） |
| 产品成本 | 库存记录（FIFO） |
| 广告费 | WB API（广告统计） |
| 其他成本 | 手动录入 |

**利润报表**：
- 单品利润明细
- 日/周/月利润汇总
- 利润趋势图表

**API 对接**：
- `GET /api/v1/supplier-report` - 订单报表
- `GET /api/v1/finance/balance` - 财务余额

### 6. 后台管理

#### 6.1 用户管理

| 功能 | 说明 |
|------|------|
| 新增用户 | 创建账号 |
| 审核用户 | 激活/禁用 |
| 重置密码 | 管理员重置 |
| 权限管理 | 模块权限 + 产品权限 |

**权限类型**：
- 模块权限：看板/广告/产品/库存/财务/后台
- 产品权限：可访问的产品列表

#### 6.2 产品管理

- 从平台导入产品（同功能3）
- 支持自定义编辑
- 每次同步只新增不覆盖

#### 6.3 店铺管理

| 功能 | 说明 |
|------|------|
| API 设置 | Token 配置 |
| 自动同步频率 | 每天/每小时 |
| 手动同步 | 立即同步 |
| 货币种类 | 人民币/卢布 |
| 编辑/删除 | 店铺管理 |

#### 6.4 系统设置

| 功能 | 说明 |
|------|------|
| 模块管理 | 启用/禁用模块 |
| 汇率设置 | 人民币 ⇄ 卢布 |
| 预警阈值 | 广告占比 > 5% 显示红色 |

## 技术要求

### 前端
- **响应式设计**：手机 + 电脑适配
- **框架**：Vue 3 + Element Plus
- **图表**：ECharts

### 后端
- **框架**：Python FastAPI
- **数据库**：SQLite / PostgreSQL
- **定时任务**：APScheduler（自动同步）

### 部署
- **方式**：Docker Compose
- **访问**：公网 IP + 端口
- **HTTPS**：可选（需域名）

### 自动化
- **数据同步**：每天自动更新
- **无需手动**：后台定时拉取

---

## MVP 优先级

**第一阶段（核心）**：
1. 用户登录
2. 店铺管理（Token 配置）
3. 产品同步
4. 销售看板

**第二阶段（财务）**：
5. 库存管理
6. 利润分析

**第三阶段（增值）**：
7. 广告分析
8. 后台管理

## 预警规则

| 条件 | 显示 |
|------|------|
| 广告占比 > 5% | 红色警告 |
| 广告占比 > 3% | 黄色提醒 |
| 库存 < 安全值 | 预警通知 |
| 利润率 < 10% | 关注提醒 |
